# Contributing

Contributions are welcome. Run `uv sync && uv run pytest` before opening a pull request.

## DCO (Recommended)

It is recommended that commits include a Developer Certificate of Origin (DCO) sign-off:

```text
Signed-off-by: Your Name <you@example.com>
```

The easiest way to do this is by using:

```bash
git commit -s
```

While not required, using DCO sign-offs helps document authorship and makes contributing easier for projects that adopt the DCO workflow.
